# 🚀 별자리 항해 (Constellation Voyage)

지구에서 출발해 진짜 별들 사이를 워프 항해하는 3D 성간 탐사 시뮬레이터.
**별자리는 '지금, 여기'에서만 성립하는 그림**이라는 것을 세 가지 스케일로 체험합니다.

- **항해하면** — 별자리는 옆에서 보면 무너집니다 (공간의 착시)
- **기다리면** — 별자리는 수만 년이 지나면 무너집니다 (시간의 착시)
- **물러나면** — 그 모든 여행조차 은하의 티끌입니다 (스케일의 착시)

> 브라우저에서 바로 실행: **[Live Demo](https://<username>.github.io/<repo>/)**

## 주요 기능

| 기능 | 설명 |
|---|---|
| 🚀 워프 항해 | 실측 3D 좌표의 별 4,545개 사이를 0.05~500 광년/초로 이동. 겉보기 등급을 현재 위치 기준으로 GPU에서 실시간 재계산 |
| ✨ 별자리 해체 | IAU 88개 별자리 선(선분 679개)이 실제 3D 좌표를 잇고 있어, 이동하면 익숙한 형태가 실시간으로 무너짐 |
| ⭐ 실체 렌더링 | 색지수→온도, 광도→반지름을 계산해 접근 시 절차 생성 표면(쌀알무늬·주연감광·코로나)으로 전환. 베텔게우스(태양 266배)와 바너드 별(태양 10%)의 도착 장면이 다름 |
| 💫 쌍성계 | 알비레오·시리우스·알파 센타우리·미자르·안타레스·프로키온·61 Cygni — 동반성 존재는 실측, 궤도 움직임은 연출 |
| ⏳ 시간 여행 | 실측 공간속도(고유운동+시선속도)로 ±50,000년 외삽. 자동 재생·역재생, 📌 별 추적 시네마틱 캠 |
| 🔭 발견 도감 | 지구에서 6등급보다 어두워 안 보이던 숨은 별 1,600여 개 — 4광년 안으로 접근하면 발견·등록 |
| 📊 H-R 다이어그램 | 방문·발견한 별이 온도-광도 그래프에 자동으로 찍히는 나만의 별 도감 |
| 🌌 은하 지도 | 은하면 투영 2단 줌 — 탐사 영역(±2,500 ly) ↔ 은하 전체(±56,000 ly). 은하수 배경은 J2000 은하좌표 변환으로 실제 방향에 렌더링 |
| 🌙 실관측 가이드 | 아무 별이나 탭 → 최적 관측 월, 남중 고도·방향(위도 37.3°N 기준), 필요 장비 안내 |
| 📓 여행 일지 | 누적 항해 거리, 방문 기록, 발견 수 — localStorage에 영구 저장 |

## 조작법

- **드래그** — 시선 회전 · **휠 / 두 손가락 핀치** — 전진·후퇴
- **별 탭** — 정체 확인 (별자리·거리·온도·크기·광도 + 이야기·찾는 법·위키)
- **하단 바** — 목적지 선택, 워프 속도 · **⋯** — 부가 기능 펼치기 · **⏳** — 시간 여행
- 5초간 입력이 없으면 UI가 자동으로 흐려집니다

## 추천 코스

1. **베가 → 데네브**: 여름의 대삼각형이 25광년 vs 1,412광년으로 찢어지는 것 확인
2. **알비레오**: 황금+사파이어 쌍성을 눈앞에서
3. **바너드 별**: 도착 → 물러남 → ⏳ 📌 추적 + ▶ 재생 — 5만 년에 23광년을 달리는 별
4. **🌌 지도 → 은하 전체**: 지금까지의 모든 여행이 푸른 티끌 하나

## 배포 / 로컬 실행

정적 단일 파일이라 별도 빌드가 없습니다.

```
1. index.html을 저장소 루트에 업로드
2. Settings → Pages → Branch: main / root → Save
```

로컬에서는 index.html을 브라우저로 열면 됩니다 (Three.js CDN 로드를 위해 인터넷 연결 필요).

## 기술

- Three.js r128 — WebGL 포인트 클라우드(커스텀 셰이더), 절차 생성 항성 표면
- 순수 HTML/JS 단일 파일 (~410KB, 별 데이터 내장), 모바일 터치 지원
- 겉보기 등급 재계산: `m = M + 5·log₁₀(d/10pc)` (버텍스 셰이더)
- 항성 반지름 추정: `R ∝ √L · (T☉/T)²` (슈테판-볼츠만)

## 데이터 출처 및 저작자 표시

- **별 카탈로그**: [HYG Database v3.7](https://github.com/astronexus/HYG-Database) (Astronexus, CC BY-SA) — 좌표·등급·색지수·고유운동 실측치
